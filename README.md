# js-os-info

> Custom tool for  os info.

#### Build Setup
## install dependencies
```javascript
npm i js-os-info
```

#### Readme

```javascript
1. import osInfo from 'js-os-info'
2. // test OS info
    osInfo.update()
3. if(osInfo.isSurface){
    console.log('it is surface')
   }
```
